Chapter 1 Prerrequisitos

  1. Instala R desde CRAN
  2. Instala RStudio desde POSIT
  3. Abre RStudio por primera vez y crea un archivo .r: File / New File / R Script
  4. Decide donde guardar el .r: File / Save as
  5. Ese .r es una ‘receta’ del código que se ejecutará en la consola de R.
  6. A partir de ahora podrás escribir código en tu .r. Para ejecutarlo selecciona la línea que quieres ejecutar y pulsa Ctrl + Enter o ejecuta todo el código de una vez en Run arriba a la derecha de la ventana donde leerás el .r

Algo fundamental es conocer el directorio de trabajo. R se ejecuta sobre una localización de tu ordenador en concreto. Si quieres importar archivos a R (como, por ejemplo, una base de datos en excel), lo mejor es que lo pongas en el mismo directorio en el que se está ejecutando R. Con el comando getwd() (“get the working directory”), podrás conocer el directorio.

getwd()

La recomendación es siempre establecer el directorio de trabajo en el mismo lugar en el que tengas guardado el archivo .r. Esto ocurre automáticamente siempre que abres R haciendo doble clic sobre un archivo .r. Si tienes algún problema puedes ir al menú superior y hacer Session / Set Working Directory / To Source File Location.

Consejo: cada vez que escribas una nueva línea de código en tu archivo .r recuerda hacer Ctrl + R para guardar los cambios.
